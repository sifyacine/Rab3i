# مِلاحة | Melaha Navigator

مِلاحة هي الشريك الإبداعي الذي يساعدك على إيصال رسائلك بشكل إبداعي يبني تصوّر ويحقق هدف.

## التقنيات المستخدمة

- React
- Vite
- Tailwind CSS
- Shadcn/UI
- Lucide React

## التشغيل المحلي

```bash
npm install
npm run dev
```

## بناء المشروع

```bash
npm run build
```

## النشر على Netlify

هذا المشروع جاهز للنشر على Netlify. يمكنك ربط المستودع بـ Netlify وسيتم البناء والنشر تلقائياً باستخدام الإعدادات الموجودة في `netlify.toml`.
